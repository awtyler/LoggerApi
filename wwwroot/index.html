Use the following endpoints to write to a log:

NOTE: All entries in the loggers are held in volatile memory and can be removed at any time.

Add to a log:

POST https://logger.example.com/Log/{logName}
Appends to a log
	BODY (JSON): 
	{
		"date": "2020-01-10T14:23:12.023"
		"text": "Logged Message"
	}

Read all lines from a log:

GET https://logger.example.com/Log/{logName}
Returns array of strings in JSON format
GET https://logger.example.com/Log/{logName}/json
Returns array of LogEntry objects in JSON format (i.e. {"date": "xxx", "text": "theText"})

List Log Names:

GET https://logger.example.com/Log/LogNames
Returns array of strings in JSON format

Get preformatted log entries, ordered by date then by message text:

GET https://logger.example.com/Log/{logName}
Returns array of strings in JSON format
