					Use the following endpoints to write to a log:
					
					NOTE: All entries in the loggers are held in volatile memory and can be removed at any time.

					Add to a log:

					POST https://logger.example.com:5001/Log?log=logName
						BODY: "LogText Goes Here"
						NOTE: Include quotes in the body


					Read all lines from a log:

					GET https://logger.example.com:5001/Log?log=logName
						Returns array of strings in JSON format


					List Log Names:

					GET https://logger.example.com:5001/Log/LogNames
						Returns array of strings in JSON format
